3-) API sonuçlarına ulaşmak için 
API'ye başka bir cihazdan erişim sağlamak için, 
API'nin çalıştığı Raspberry Pi'nin IP adresini ve API'nin endpoint'ini kullanarak API'ye 
HTTP isteği göndermeniz gerekiyor. Bu işlem için bir web tarayıcısı veya bir 
programlama dili (örneğin, Python) kullanabilirsiniz.
Örneğin, http://<Raspberry Pi IP adresi>:<port>/vibration-data şeklinde bir istek göndererek,
API'ye erişim sağlayabilirsiniz. Burada <Raspberry Pi IP adresi> Raspberry Pi'nin IP adresini,
<port> ise Flask uygulamasının hangi portta çalıştığını belirtir. Örneğin, eğer Flask uygulamasını
varsayılan port olan 5000 numaralı portta çalıştırıyorsanız, <port> parametresini 
5000 olarak ayarlamanız gerekir.

Bu isteği gönderdikten sonra, API'nin JSON formatında titreşim verilerini döndürmesi gerekir. Bu verileri programınızda kullanabilir veya web tarayıcısında görüntüleyebilirsiniz.

Not: Raspberry Pi'nin IP adresini öğrenmek için, Raspberry Pi terminalinde ifconfig komutunu çalıştırabilirsiniz.
